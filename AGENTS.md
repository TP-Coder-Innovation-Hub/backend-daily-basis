# AGENTS.md

## Build & Test Commands
- Build: `./mvnw clean package` or `./gradlew build`
- Test: `./mvnw test` or `./gradlew test`
- Run: `./mvnw spring-boot:run` or `./gradlew bootRun`

## Project Conventions
- Java 17+ with Spring Boot 3.x
- All code examples use Spring Boot with real dependencies
- Follow existing code style in each module
- Each topic is self-contained with step-by-step explanations
- Mermaid diagrams where architectural concepts need visualization

## Content Guidelines
- 200-500 words per file
- SHORT, CONCISE, DIRECT
- No preamble or filler
- Real Spring Boot Java code with step-by-step explanation
- No emojis, no level badges
